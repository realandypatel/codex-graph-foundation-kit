# Support

## Discussions
Use GitHub Discussions for:
- setup help
- workflow questions
- Codex / MCP usage questions
- architecture questions

## Issues
Use GitHub Issues for:
- reproducible bugs
- missing or incorrect docs
- broken scripts
- concrete improvement requests

## Please include
- what you tried
- what happened
- what you expected
- environment details
- screenshots or logs when useful

Do not post secrets or credentials publicly.
