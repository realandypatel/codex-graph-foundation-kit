# Support

## Where to get help

### Discussions
Use **GitHub Discussions** for:
- setup questions
- workflow questions
- Codex / MCP usage questions
- “how do I use this in my repo?” questions

### Issues
Use **GitHub Issues** for:
- reproducible bugs
- broken scripts
- missing or incorrect documentation
- concrete improvement requests

## Before posting
Please check:
- `README.md`
- `docs/INSTALL.md`
- `docs/COMPATIBILITY.md`
- `docs/SAFE_CLAIMS.md`
- existing Discussions / Issues

## What to include
Please include:
- what you were trying to do
- what repo or environment you used
- what happened
- what you expected
- any error output or screenshots that help

## Security
Do not post secrets, tokens, or sensitive credentials in issues or discussions.

If your concern is security-sensitive, use a private contact path rather than posting publicly.
