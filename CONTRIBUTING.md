# Contributing to Codex Graph Foundation Kit

Thanks for your interest in improving this project.

## Before you contribute
Please read:
- `README.md`
- `docs/INSTALL.md`
- `docs/COMPATIBILITY.md`
- `docs/SAFE_CLAIMS.md`

## Best ways to contribute
We welcome:
- documentation improvements
- clearer setup steps
- safer public positioning / wording
- better prompt templates
- issue templates and community health improvements
- examples of real-world usage
- lightweight scripts that improve setup verification

## Before opening an issue
Please check:
- the README
- the Discussions tab
- existing issues and releases

Use Discussions for:
- setup questions
- workflow questions
- “how should I use this?” questions

Use Issues for:
- broken instructions
- missing files
- reproducible bugs
- clear improvement requests

## Pull request guidelines
Please keep pull requests:
- focused
- small when possible
- clearly described
- aligned with the repo’s purpose as a foundation kit

When opening a PR:
1. Explain the problem.
2. Explain the change.
3. Mention any docs or prompts affected.
4. Include before/after examples if helpful.

## Scope guardrails
This repository is a **foundation kit**, not the upstream graph engine.

Please avoid PRs that:
- claim this repo is the engine itself
- overstate benchmark or token reduction claims
- add unnecessary complexity without clear benefit
- turn the repo into a bloated framework

## Style preferences
- Keep wording clear and practical.
- Prefer copy-paste-ready examples.
- Keep prompts concise and operator-friendly.
- Use honest, supportable claims.

## Questions
If you are unsure where something belongs, start a Discussion first.
