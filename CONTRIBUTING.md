# Contributing

Thanks for contributing to this project.

## Before you start
Please read:
- `README.md`
- `PROJECT_CONTEXT.md`
- `SUPPORT.md`

## Use Discussions for
- setup questions
- usage questions
- architecture questions
- “where should this change go?” questions

## Use Issues for
- reproducible bugs
- clear improvement requests
- documentation errors
- broken scripts or setup steps

## Pull requests
Please keep PRs:
- focused
- clearly described
- scoped to one change when possible

When opening a PR, include:
1. What changed
2. Why it changed
3. Any files/docs/prompts affected
4. How it was verified
