# Code of Conduct

## Our pledge
We want this project to be welcoming, practical, and respectful for everyone.

## Expected behavior
Please:
- be respectful and constructive
- assume good intent
- give actionable feedback
- keep discussions relevant to the project
- help newcomers when possible

## Unacceptable behavior
This includes:
- harassment or personal attacks
- hateful or discriminatory language
- spam or deliberate disruption
- repeated bad-faith arguing
- sharing private or sensitive information without consent

## Scope
This code of conduct applies in:
- issues
- pull requests
- discussions
- other project communication spaces

## Enforcement
Project maintainers may edit, remove, lock, or close content that violates this code of conduct.

## Reporting
For support questions, use Discussions.
For sensitive concerns, use the contact/support path described in `SUPPORT.md`.
